We will use the labeled data $(X_{train}, y_{train})$ and the unlabeled data $X_{test}$ to classify the data. That is, we will treat it as a semi-supervised learning problem without using the labels of the testing data $y_{test}$.

  1. Please use self training K-means method to predict the labels of the unlabeled data $X_{test}$ through the following steps:

    • For the labeled data $(X_{train}, y_{train})$, compute the centroid (mean) of each group.
    • Find the pseudo-labels of each unlabeled data according to K-means, that is, assign the label according to the closest centroid.
    • Update the centroid of each cluster by using all the labels and pseudo-labels.
    • Use the updated centroids to assign new pseudo-labels for the unlabeled data. Do not update the true labels of the labeled data.
    • Repeat the process until convergence.

  2. We need the following outputs for the final result obtained in 1.

    • The centroid (mean) and variance matrix of each cluster.
    • The percentage of points in each cluster.
    • The prediction accuracy by comparing the predicted labels and the true labels $y_{test}$ of the testing data.
    • Plot all the points by using colors to distinguish the clusters and marker styles to distinguish the labeled points and unlabeled points.

  3. Use semi-supervised GMM method with 3 classes to predict the labels of the unlabeled data $X_{test}$ with the following instructions:

    • Use the means, variance matrices, and the percentages obtained in 2 as the initial values.
    • Run the iterations until convergence.
    • Compute $\gamma_{j,l}$, the probability of each unlabeled point $x_j$ in each cluster $l$ and assign the labels corresponding to the largest probability.
    • Compute the prediction accuracy by comparing the predicted labels and the true labels $y_{test}$ of the testing data.
    • Plot all the points by using colors to distinguish the clusters and marker styles to distinguish the labeled points and unlabeled points.

iter = 0;
# run GMM
error=1;
while (error>=1e-10):# exponential decay...slow at higher iteration
    print("iteration number {0} with error {1}".format(iter,error))
    
    
    likelihood_labeled = Y_new.copy();
    likelihood_unlabeled = []
  # Expectation step

    for j in range(class_num):
        likelihood_unlabeled.append(multivariate_normal.pdf(x=X_test, mean=Ctrd[j], cov=cov[j]))
    likelihood_unlabeled = np.array(likelihood_unlabeled).T
    
    #combine the likelihood
    likelihood= np.vstack([likelihood_labeled,likelihood_unlabeled]).T
    
    assert likelihood.shape == (class_num, len(X_self_learning))
    
    b = []
    # Maximization step 
    Ctrd_old=Ctrd.copy() # store the old centroids
    for j in range(class_num):
        #print(j)
        # use the current values for the parameters to evaluate the posterior
        # probabilities of the data to have been generanted by each gaussian
        b.append((likelihood[j] * weights[j]) / (np.sum([likelihood[i] * weights[i] for i in range(class_num)], axis=0)))
    
        # update mean and variance
        Ctrd[j] = np.sum(b[j].reshape(len(X_self_learning),1) * X_self_learning, axis=0) / (np.sum(b[j]))
        cov[j] = np.dot((b[j].reshape(len(X_self_learning),1) * (X_self_learning - Ctrd[j])).T, (X_self_learning - Ctrd[j])) / (np.sum(b[j]))

        # update the weights
        weights[j] = np.mean(b[j])

        assert cov.shape == (class_num, X_self_learning.shape[1], X_self_learning.shape[1])
        assert Ctrd.shape == (class_num, X_self_learning.shape[1])
    error=LA.norm(Ctrd_old-Ctrd)
    iter = iter+1;
